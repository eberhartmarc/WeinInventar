# WeinInventar – AppSheet Vorlage

Dieses Repository enthält eine direkt nutzbare Vorlage, um eine Weinkeller-App mit Google AppSheet auf Basis einer einzigen Tabelle `Weine` aufzubauen.

## Inhalt

- `APPSHEET_SETUP.md` – Schritt-für-Schritt-Konfiguration der App (Spalten, Formeln, Views, Aktionen).
- `Weine.csv` – Tabellenstruktur mit allen geforderten Spalten sowie 5 Beispiel-Datensätzen.

## Schnellstart

1. Neues Google Sheet erstellen.
2. Inhalt von `Weine.csv` in ein Tabellenblatt mit dem Namen `Weine` kopieren.
3. In AppSheet eine neue App auf Basis dieses Sheets anlegen.
4. Die Konfigurationsschritte aus `APPSHEET_SETUP.md` umsetzen.
